Welcome to the PokePlush Paradise Database!

PokePlush Paradise sells high-quality Pokémon plushies for fans and collectors alike. Our expansive collection features Pokemon from each generation, resulting in more than 1,000 different types of plushies.

Page Descriptions

Webpage Description
Home Overview of the site.
Customers Ability to add new Customers, indicated by first and last name, email, and home address. Able to edit existing Customers or delete listed Customers.
Pokemon Ability to add new Pokémon plushies, indicating name, type, price, and stock quantity. Able to edit existing plushies or delete listed plushies.
Sales Ability to add sales completed by customers on indicated sale date using custID and date of sale. Also able to edit and delete existing sales.
SaleItems Ability to add individual sales items using saleID, plushieID and quantity of plushies purchased. Also able to edit and delete existing sale items.
